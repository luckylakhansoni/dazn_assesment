# Node Rest API

This project is a REST API with Node.js and Express.

## Prerequisites
- Node.js 10+
- NPM

## Installation
- Install dependencies
```bash
npm install
```
- Create local environment file
```shell
cp .env.example .env
nano .env
```
please mention above your local environment credentials

- Start Application
```bash
npm start
```

- Run test with chai mocha
```bash
npm test
```

- PostMan collection
```bash
attched
```
